## 주요 기능
- 하루 계획을 세워 계획을 달성하면 코인처럼 별을 획득할 수 있다.
- 획득한 별로 Shop에서 행성을 구매해 자신만의 은하를 채워나갈 수 있다.
- 어제보다 더 나은 '나'로 발전할 수 있다.
